ArkromePro
Tweaks

Supports iOS 12 & 13
Compatible with A12 & A13
A customizable version of Arkrome with new features!
Features:

  • Very light-weight option to show you the percentage while keeping the battery icon.
  • An option to hide the stock CC percentage.
  • An option to hide the charging indicator.
  • Disable it while charging.
  • Optimizes the colorization based on the background color.
  • Dynamic colors based on current percentage.
  • Battery animation.
  • Double tap to respring option.
  • Disable the low power mode yellow color.
  • Set custom colors when charging and when NOT charging.
  • Configure percentage color.
  • Configure body color.
  • Configure pin color.
  • Apply colors to the body only, the percentage only or both.
  • Supports the old statusbar (pre iPhone X devices) on iOS 13.
  • Supports the iPad statusbar.
  • Supports RTL languages.

Experimental features:

  • Option to enable the modern status bar for devices that don't have it.

Notes:
It does NOT support iOS 11.
